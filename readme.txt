Dashboard widget showing all the latest interviews from Shardul Pandey Talks.

== Description ==


The plugin activates a Dashboard Widgets that shows up latest interviews from Shardul Pandey Talks. Shardul Pandey Talks (http://sangkrit.net/talks) is international program for promoting those individuals, who register their domain to create virtual wealth in webspace so if you too want to get interviewed, simply email to shardulpandey@sangkrit.org considering any conversation upon this as subject to publication.

== Installation ==
Simply install and activate it. Now visit Dashboard -> Home for reading recent interviews from Shardul Pandey Talks.
